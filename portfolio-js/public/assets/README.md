# Project Assets Guide

Place your project screenshot images in this folder with the following names:

- `finscope.png` - FinScope Real-Time Analytics Dashboard
- `donorhub.png` - DonorHub Charity & Donation Platform
- `hirelens.png` - HireLens Intelligent Resume Intelligence
- `nextrack.png` - NexTrack Inventory Management System
- `loanverse.png` - LoanVerse Loan Management System
- `nexagen.png` - NexAgen AI-Powered Meeting Scheduler

**Recommended image specs:**
- Dimensions: 1200x800px or similar (will be cropped to fit)
- Format: PNG
- Content: Show the actual application UI/dashboard

---

## Videos Folder (`/assets/videos/`)

Place your demo video files in the `videos` subfolder:

- `finscope.mp4`
- `donorhub.mp4`
- `hirelens.mp4`
- `nextrack.mp4`
- `loanverse.mp4`
- `nexagen.mp4`

**Recommended video specs:**
- Format: MP4 (H.264 codec)
- Resolution: 1920x1080 or lower
- Duration: 30-60 seconds recommended
- Size: Keep under 10MB each for fast loading

---

**Note:** The Projects component has fallback placeholders if images are missing, so the site will still look good even without actual assets.
